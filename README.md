# MSI-B460M-i5-10400-Hackintosh

My personal OpenCore hackintosh for MSI B460M i5-10400 UHD 630.

## Update Log

- OC version: 0.7.8
- Latest update: 2022-03-09

## Features

- Concisely EFI
- Reconfig USB ports
- Support macOS Monterey
